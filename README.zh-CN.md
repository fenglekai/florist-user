# 用户模块

> 后台用户服务模块，只提供基础的数据库、用户登录/注册功能

## 快速入门

如需进一步了解，参见 [midway 文档][midway]。

### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7002/
```

### 部署

```bash
$ npm start
```

### 内置指令

- 使用 `npm run lint` 来做代码风格检查。
- 使用 `npm test` 来执行单元测试。


[midway]: https://midwayjs.org

### 插件

- bcrypt
- uuid
- jwt
- grpc
